Design Decisions

DD 20100521: The factory method of DataDescriptor requires type argument in addition to data argument

Two reason by now:

  • In the case of the given data is null, client is allowed to specify the type explicitly.
  • The type argument provides possibility to support some kind of explicit declared type instead of using the actual type of the data.
